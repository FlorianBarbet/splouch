# Splouch
Experimental nodejs home-made framework
 
# First Installation
```shell
npm i -g nodemon typescript ts-node path
```
# Install

```shell
npm ci
```

# Run
```shell
npm run start 
# OR
npm run dev #dev mode obviously
```
